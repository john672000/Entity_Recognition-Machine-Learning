# Entity_Recognition-Emergency-Database-
This application is built using flutter as the front framework and Python FastAPI for the Backend. I've generated a random 911 call interactions using Python and have BIO-TAGGED them to mark my entities. Later we train a BERT model to understand the tags pattern and we use this model to pass a conversation that is being streamed from audio to text. We use OpenAI's whisper base model for faster transcription and based on the conversation that is being streamed we argument it to the BERT Model for the extraction and this happens in stream too as we cannot afford any latency for a model of this kind. Yet to integrate more modules which provides a location based on the location tag found and the possibiltes are infinite. Will continue working on this in the future too...
